## üõ†Ô∏è Installation

### Prerequisites
- **Android Studio**: Version Giraffe (2022.3.1) or later
- **SDK Versions**: Min SDK 24 (Android 7.0), Target SDK 34
- **Java**: Version 8 or higher
- **VR-Ready Device**: Cardboard, Daydream, or compatible headset (optional for handheld mode)
- **Sensors**: Gyroscope required for full VR immersion

### Setup Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/ramzy-ahmed/Movies-VR-App.git
   ```
2. Open the project in Android Studio.
3. Sync Gradle dependencies.
4. Enable VR mode in `AndroidManifest.xml` (if using Google VR SDK or Cardboard).
5. Build and run:
   ```bash
   # From Android Studio: Click Run
   ```

> **Note**: For full VR, pair with a compatible headset. Handheld mode works on any device with gyroscope.
